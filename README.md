# DF FN Tools

[![Blender Version](https://img.shields.io/badge/Blender-4.4+-blue.svg)](https://blender.org)
[![License](https://img.shields.io/github/license/Aloija/DF_FN_Tools)](LICENSE)

**DF FN Tools** — это аддон для Blender, предназначенный для упрощения экспорта 3D-моделей в формате FBX. Аддон разработан специально для работы с Unreal Engine и поддерживает структуру именования LOD'ов, UCX-коллизий и Nanite-геометрии.

## 📋 Особенности

- **Автоматический экспорт в FBX** с предустановленными настройками для Unreal Engine
- **Поддержка уровней детализации (LOD)**: LOD0, LOD1, LOD2, LOD3
- **Экспорт Nanite-геометрии**: SM_NITE_* префикс
- **Коллизии UCX**: автоматическое объединение с основным мешем
- **Умное обнаружение связанных мешей** по базовому имени
- **Автоматическая переименовка UV-каналов**: map1 и uvSet1
- **Переназначение материалов** в экспортируемых мешах
- **Сброс трансформаций** перед экспортом (опционально)
- **Однокликовое обновление** аддона из GitHub

## 🚀 Установка

### Стандартная установка

1. Откройте Blender 4.4+
2. Перейдите в `Edit > Preferences > Add-ons`
3. Нажмите `Install...` и выберите файл `.zip` с репозитория или `.py` файл
4. Найдите `DF FN Tools` в списке и включите галочку

### Установка из репозитория

```bash
git clone https://github.com/Aloija/DF_FN_Tools.git
```

Затем установите как обычный аддон Blender.

## 🔧 Использование

### Панель экспорта

Аддон добавляет панель экспорта в `TOPBAR`:

![UI](https://placehold.co/600x200/2a2a2a/white?text=Export+Panel+Preview)

#### Основные настройки

| Настройка | Описание |
|-----------|----------|
| **Path** | Путь к папке экспорта (относительный или абсолютный) |
| **Reset transforms** | Сброс трансформаций (Loc/Rot/Scale) перед экспортом |
| **Export with LODs and UCX** | Автоматический поиск и экспорт связанных мешей |
| **Apply single material** | Применить один материал ко всем мешам |
| **Material name** | Имя материала для применения |
| **Reassigne materials** | Включить ручное переназначение материалов |

### Структура именования

Для правильной работы аддона используйте следующую нотацию:

| Пример | Тип | Описание |
|--------|-----|----------|
| `LOD0_SM_MyMesh` | Меш основного уровня | Базовая геометрия |
| `LOD1_SM_MyMesh` | LOD 1 | Первый уровень детализации |
| `LOD2_SM_MyMesh` | LOD 2 | Второй уровень детализации |
| `LOD3_SM_MyMesh` | LOD 3 | Третий уровень детализации |
| `SM_NITE_MyMesh` | Nanite | Геометрия для Nanite (UE5) |
| `UCX_LOD0_SM_Door_01` | UCX | Упрощенная коллизия |

### Контекстное меню

В контекстном меню объектов добавлены операторы:

- **Clean Join** — Дублирует, применяет модификаторы и объединяет выделенные мешей
- **Rename UVs** — Переименовывает UV-каналы в `map1` и `uvSet1`

### Панель экспорта (popover)

В строке редактора отображается:

- Кнопка быстрого экспорта
- Счетчик вершин (актуальный при `Shade Smooth`)
- Кнопка автообновления счетчика

## ⚙️ Настройки аддона

Откройте через `Edit > Preferences > Add-ons > DF FN Tools`:

```
DF FN Tools
├── GitHub Update
    └── [Update from GitHub] — Скачать и установить последнюю версию
```

```
DF_FN_Tools/
├── __init__.py           # Регистрация аддона и настройки
├── export.py             # Основная логика экспорта FBX
├── handlers.py           # Обработчики событий (загрузка, обновление)
├── mesh_object_classes.py # Классы данных для работы с мешами
├── ui.py                 # Интерфейс и панели
├── updater.py            # Автоматическое обновление из GitHub
└── utils.py              # Вспомогательные функции
```
